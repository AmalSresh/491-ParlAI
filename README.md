# 491-ParlAI

ParlAI is a full-stack sports betting web-app that uses vitual currency and is hosted through Microsoft Azure.

## Frontend
React  
  React-router  
  Vite  

## Backend
  Microsoft Azure  
  Azure Static Web Apps  
  Microsoft Entra ID  
  GitHub Actions  

## Prerequisites to run on your environment
  Node.js  
  npm  
  Git  
  An Azure account (If deploying)  


## To build and test the app 

  Clone  the repository:  
  ```git clone https://github.com/AmalSresh/491-ParlAI.git```

  navigate to the frontend directory  
    ```cd frontend```  

Install dependencies  
  ```npm install```
    
  In one terminal run  
  ```npm run dev```

Open another terminal and run:  
  ```npm swa start http://localhost:5173```    
  
  This will start the Azure emulator to simulate authentication. Click the second localhost link to use it.
